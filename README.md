## WooCommerce Middleware Layer | Website Automation with Python
In this project, we will create a Python script that automates the process of creating a WooCommerce product, adding a product to a WooCommerce cart, and checking out the product using the WooCommerce API.

## Installation
To install the required dependencies, run the following command:

```bash
pip install -r requirements.txt
```

## Usage
To run the script, execute the following command:

```bash
python automation.py
```
